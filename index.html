First Month Training Plan for 10K Run

Day Activity
Monday Rest or light stretching/yoga
Tuesday Easy run (3 km, 9:30-10:00/km)
Wednesday Rest or foot strengthening exercises
Thursday Easy run (3-4 km, 9:30-10:00/km)
Friday Rest
Saturday Longer run (start 4 km, increase 0.5 km weekly; 9:30-10:00/km)
Sunday Cross-training (30 min cycling/swimming)
